## 🛠️ Optimizing your files

The main dashboard lists your files. Select a file to see options.

*   **Deduplication:** The tool compares text blocks. It removes identical or repeated instructions. This prevents errors in logic.
*   **Misplacement Detection:** Sometimes instructions belong in a global profile rather than a project file. The app suggests moving these instructions to the correct level.
*   **Compression:** Long prompts create bloat. This feature removes whitespace and unnecessary comments. This keeps your instructions beneath the token limit.
*   **Token Visualization:** View usage graphs for different scopes. See how much of your total limit remains.

## ❓ Troubleshooting

**The application does not start.**
Make sure you run the installer as an administrator. Right-click the installer file and select "Run as administrator."

**The app reports no CLAUDE.md files found.**
Ensure your files have the exact name `CLAUDE.md`. The scan looks for the specific character set. Check that your folders are not hidden.

**Token counts seem off.**
Token counting depends on the specific AI model chosen in the settings menu. Open "Settings" and ensure the model matches the one you use.

**Changes caused a conflict.**
Use the "Undo" button on the main toolbar. If that fails, go to the "backups" folder inside your project. Copy the old file over the new one to restore your state.
